# Beginner-Friendly Habit Tracker

A simple  single-page web-based app to easily create habits and track them for 7-day 

## How to Run Locally
1. Download or clone this project folder to your computer.
2. Locate the file named `index.html`.
3. Simply double-click the `index.html` file to instantly launch the tracking workspace in your web browser. No need of complex terminal software commands or downloads.
